AccessiScan AI is an AI-powered accessibility testing tool that helps developers identify and resolve accessibility issues in websites. It evaluates key aspects such as color contrast, page structure, and keyboard navigation, and provides intelligent recommendations for improvement. With its web platform and browser extension, AccessiScan AI simplifies accessibility auditing and helps create more inclusive and user-friendly digital experiences. Key Features:- Automated accessibility issue detection AI-powered remediation recommendations Color contrast and readability analysis ARIA label and attribute validation Heading hierarchy and structure analysis Semantic HTML and landmark verification Keyboard navigation assessment Accessibility simulation and testing tools Browser extension for real-time accessibility insights Detailed accessibility reports and actionable feedback Technology Stack:- Frontend Next.js React TypeScript Backend Node.js Express.js Database MongoDB Artificial Intelligence:- Ollama Vision Language Models (VLMs) Project Structure:- client/ Frontend application built with Next.js and React server/ Backend services and REST APIs extension/ Browser extension for real-time accessibility analysis

Mission:-

AccessiScan AI aims to make the web more accessible by providing intelligent tools that help developers detect accessibility issues early, implement effective fixes, and ensure compliance with modern accessibility standards.

Setup & Installation:-

Setup Instructions:-

Prerequisites:-

Node.js (v18 or later) npm or yarn MongoDB Ollama (for AI integration)

Installation:-

Clone the repository to your local machine. Open the project folder. Install dependencies for both frontend and backend using npm or yarn. Configuration Create environment configuration files. Add MongoDB connection string. Set Ollama API endpoint and other required variables. Running the Project Start MongoDB service. Run the backend server. Run the frontend development server. Open the project in your browser using the local development URL. Usage Enter a website URL in the input field. Start the accessibility scan. View detected issues and AI suggestions. Apply improvements to enhance accessibility.

Development Mode:-

Run frontend and backend together to test changes and develop new features efficiently.

Authors Developed by Rishika Abrol and Team.

Demo Instructions:-

Open the AccessiScan AI website in your browser. On the homepage, enter the URL of any website you want to test for accessibility. Click on the Scan button to start the analysis. Wait for the system to process the page and generate results. Review the accessibility report, which includes:

Detected issues Severity levels AI-generated suggestions for fixes Apply the recommended improvements to understand how accessibility can be enhanced. Re-scan the updated website to see improvements in the results.

This demonstrates how AccessiScan AI identifies accessibility problems and helps improve web usability in real time.
